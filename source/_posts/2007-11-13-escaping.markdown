This is a handy code that wraps a string with CDATA escaping any CDATA inside it:

```c#
public static string CData(string data)
{
    if (string.IsNullOrEmpty(data))
    {
        return string.Empty;
    }

    string result = Regex.Replace(data, "]]>", "]]]]><![CDATA[>");
    return "<![CDATA[" + result + "]]>";
}
```
